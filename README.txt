# Descargador de Imágenes de Libros CONALITEG

Aviso
Este script está destinado a fines educativos y de aprendizaje. Asegúrate de cumplir con los términos de uso del sitio web de CONALITEG
antes de descargar y utilizar su contenido.

Este script de Python es un ejercicio escolar que permite descargar imágenes de libros del sitio web de CONALITEG
y organizarlas en carpetas según la nomenclatura de cada libro.

## Requisitos

- Python 3.x
- Bibliotecas: `requests`, `beautifulsoup4`, `urllib3`

Instala las bibliotecas necesarias con el siguiente comando:

```bash
pip install -r requirements.txt

Uso

1. Clona o descarga este repositorio en tu máquina local.
2. Abre una terminal y navega hasta el directorio del repositorio.
3. Ejecuta el script 'script.py'

El script realizará las siguientes acciones:

- Creará las carpetas "libros" y "temporal" si no existen.
- Obtendrá enlaces a las páginas de libros del sitio web de CONALITEG.
- Descargará imágenes de los libros y las guardará en la carpeta "temporal".
- Organizará las imágenes descargadas en subcarpetas dentro de la carpeta "libros".
